Class PatternDetector with:
- `__init__(self, storage: AgentStorage)`: store storage reference
- `async_detect_patterns(self) -> list[dict]`:
  1. Query events from last 14 days grouped by entity_id, domain, service, day_of_week, hour with HAVING COUNT(*) >= 3 (per D-03, D-04):
  ```sql
  SELECT entity_id, domain, service, day_of_week, hour,
         COUNT(*) as occurrences, MAX(timestamp) as last_seen
  FROM events
  WHERE timestamp >= datetime('now', '-14 days')
  GROUP BY entity_id, domain, service, day_of_week, hour
  HAVING COUNT(*) >= 3
  ORDER BY occurrences DESC
  ```
  2. For each result row, upsert into patterns table:
  ```sql
  INSERT INTO patterns (entity_id, domain, service, day_of_week, hour, occurrences, last_seen, detected_at)
  VALUES (?, ?, ?, ?, ?, ?, ?, ?)
  ON CONFLICT(entity_id, service, day_of_week, hour)
  DO UPDATE SET occurrences=excluded.occurrences, last_seen=excluded.last_seen, detected_at=excluded.detected_at
  ```
  3. Commit and return list of dicts with pattern data
- `async_get_patterns(self) -> list[dict]`: SELECT * FROM patterns, return as list of dicts

No imports of httpx, requests, aiohttp, urllib.
