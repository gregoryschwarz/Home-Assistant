**Step 4: Create custom_components/ha_ai_agent/storage.py** (GREEN phase). Implement AgentStorage class:

```python
class AgentStorage:
    def __init__(self, hass: HomeAssistant) -> None:
        self.hass = hass
        self._db: aiosqlite.Connection | None = None
        self._db_path = os.path.join(hass.config.config_dir, DB_FILENAME)
```

Methods:
- `async_open()`: connect via `aiosqlite.connect(self._db_path)`, execute `PRAGMA journal_mode=WAL`, `PRAGMA synchronous=NORMAL`, commit, call `_ensure_schema()`
- `async_close()`: close db if not None, set to None (idempotent)
- `_ensure_schema()`: CREATE TABLE IF NOT EXISTS for `meta` (key TEXT PK, value TEXT) and `events` (id INTEGER PK AUTOINCREMENT, entity_id TEXT NOT NULL, domain TEXT NOT NULL, service TEXT NOT NULL, timestamp TEXT NOT NULL, day_of_week INTEGER NOT NULL, hour INTEGER NOT NULL, persons_home TEXT, weather_condition TEXT). INSERT OR IGNORE schema_version=1 into meta. CREATE INDEX IF NOT EXISTS idx_events_entity_ts ON events(entity_id, timestamp). Commit.
- `async_record_event(entity_id, domain, service, persons_home, weather_condition)`: INSERT into events with `datetime.now(timezone.utc).isoformat()` for timestamp, `.weekday()` for day_of_week (0=Monday per D-07), `.hour` for hour, `json.dumps(persons_home)` for persons_home. Then call `async_enforce_cap()`. Commit.
- `async_purge_old_events()`: `DELETE FROM events WHERE timestamp < datetime('now', '-{HABIT_TTL_DAYS} days')`. Commit.
- `async_enforce_cap()`: `DELETE FROM events WHERE id IN (SELECT id FROM events ORDER BY id ASC LIMIT MAX(0, (SELECT COUNT(*) FROM events) - {HABIT_EVENT_CAP}))`. Commit.

Import only: `aiosqlite`, `os`, `json`, `logging`, `datetime` from stdlib. No httpx/requests/aiohttp/urllib.
