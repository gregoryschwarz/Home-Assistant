```python
async def async_setup_entry(hass: HomeAssistant, entry: ConfigEntry) -> bool:
    hass.data.setdefault(DOMAIN, {})
    allowed_domains = entry.options.get(CONF_ALLOWED_DOMAINS, DEFAULT_ALLOWED_DOMAINS)
    router = IntentRouter(hass, allowed_domains=allowed_domains)
    entity_context = EntityContextBuilder(hass, allowed_domains=allowed_domains)
    hass.data[DOMAIN][entry.entry_id] = {
        "entry": entry,
        "router": router,
        "entity_context": entity_context,
    }
    await hass.config_entries.async_forward_entry_setups(entry, PLATFORMS)
    return True
```

**Step 2 — Create `claude_client.py`** following the full skeleton from RESEARCH.md (Pattern 1-8):

- `from __future__ import annotations` at top
- `_LOGGER = logging.getLogger(__name__)`
- Class `ClaudeClient` with `__init__(self, hass, api_key, allowed_domains)`:
  - `self.hass = hass`
  - `self._allowed_domains = allowed_domains`
  - `self._client = AsyncAnthropic(api_key=api_key, timeout=10.0, max_retries=0)` (D-04, D-07)
  - `self._history: deque = deque(maxlen=MAX_HISTORY_TURNS)` (D-08)
- Method `async_complete(self, text: str, entities: list[dict]) -> str`:
  - Format entity list as string: `"- {entity_id} ({friendly_name}, etat: {state})"` per entity
  - Build user_content = `f"{text}\n\nEntites disponibles:\n{entity_list_str}"`
  - Append `{"role": "user", "content": user_content}` to history
  - Format system prompt with `SYSTEM_PROMPT.format(allowed_domains=", ".join(self._allowed_domains))`
  - Retry loop: `for attempt in range(2)` with try/except (Pattern 3 from RESEARCH):
    - `AuthenticationError` -> pop last history entry, log WARNING, return D-19 string
    - `(APIConnectionError, APITimeoutError)` -> if attempt==0: `asyncio.sleep(1)` + continue; else pop history, log WARNING, return D-18 string
    - `APIStatusError` -> pop history, log WARNING, return D-18 string
  - Call `self._handle_response(response, text)` on success
- Method `_handle_response(self, response, original_text: str) -> str`:
  - If `response.stop_reason == "tool_use"`: extract tool_block, validate domain against `self._allowed_domains` (D-11), call `_execute_service`, store text-only in history (Pitfall 1)
  - If `response.stop_reason == "end_turn"`: extract text block, store in history, return text (D-12)
  - Otherwise return None
- Method `_execute_service(self, tool_input: dict, original_text: str) -> str`:
  - Extract domain, service, entity_id, service_data from tool_input
  - `service_data["entity_id"] = entity_id`
  - `await self.hass.services.async_call(domain, service, service_data, blocking=True)`
  - Catch `ServiceNotFound` -> return French error
  - Catch `HomeAssistantError` -> return French error
  - Return `f"D'accord, j'ai effectue l'action sur {entity_id}."`
- Method `async_close(self) -> None`:
  - `await self._client.close()` (cleanup for unload per D-09)
